# Stadion 24/7

O'zbekistonda futbol maydonlarini onlayn band qilish tizimi.

## Features

- Onlayn band qilish
- Maydonlarni ko'rish va tanlash
- Foydalanuvchi uchun qulay interfeys (Uzbek/Russian)
- "Lovable" dizayn estetikasi

## Tech Stack

- React + Vite
- TypeScript
- Tailwind CSS (Modern Green Theme)
- shadcn/ui

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
